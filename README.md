# 🧮 Modern Web Calculator

A sleek and responsive **web-based calculator** built using **HTML, CSS, and JavaScript**.
This project demonstrates how basic web technologies can be used to create an **interactive, modern, and mobile-friendly calculator** that runs directly in any browser.

## 🚀 Live Demo

Try the calculator here:

🔗 https://highorder-thinking.github.io/Calculator/

## ✨ Features

✔ Clean and modern UI design
✔ Works on **mobile and desktop browsers**
✔ Supports basic arithmetic operations
✔ Power operator **(^)** support
✔ Square root calculations
✔ Parentheses **( )** support
✔ Smooth button interactions
✔ Responsive layout for different screen sizes

## 🛠️ Built With

This project is built using:

* **HTML5** → Structure of the calculator
* **CSS3** → Styling and responsive layout
* **JavaScript** → Calculation logic and interactions

These technologies allow the calculator to run **directly in the browser without any external frameworks**.

## 📂 Project Structure

Calculator
│
├── index.html → Main calculator interface
├── style.css → Styling and responsive design
├── script.js → Calculator logic
└── README.md → Project documentation

## 📱 Browser Compatibility

This calculator works on:

* Google Chrome
* Mozilla Firefox
* Microsoft Edge
* Safari
* Mobile browsers (Android & iOS)

Since it runs fully in the browser, **no installation is required**.
